🚀 HOW TO RUN
-------------
1. Double-click "index.html" OR
2. Right-click "index.html" → Open with → Your Browser OR
3. Drag "index.html" into your browser window

That's it! No installation, no build, no server needed!


🎮 QUICK CONTROLS
-----------------
Mouse/Touch:
  • Click dice to roll
  • Click token to move

Keyboard:
  • Space/Enter = Roll dice
  • Tab = Navigate
  • Enter = Activate


⚙️ QUICK SETTINGS
------------------
• Dark Mode: Settings → Toggle Dark Mode
• Sound: Settings → Toggle Sound Effects
• Speed: Settings → Animation Speed (Fast/Normal/Slow)


💾 SAVE/LOAD
------------
• Save: Click 💾 icon during game
• Load: Main Menu → Resume Game


🎯 QUICK RULES
---------------
1. Roll 6 to exit base
2. Rolling 6 = extra turn
3. Land on opponent = capture (they go back to base)
4. Safe squares = starting positions + home lanes
5. Exact roll needed to finish
6. First to finish all 4 tokens wins!


🎨 GAME MODES

🐛 TROUBLESHOOTING
------------------
Problem: Game won't load
→ Enable JavaScript in browser
→ Try different browser
→ Check console (F12) for errors

Problem: Can't save game
→ Exit private/incognito mode
→ Allow localStorage in browser settings

Problem: Board not showing
→ Update browser to latest version
→ Enable hardware acceleration
→ Resize window to refresh
